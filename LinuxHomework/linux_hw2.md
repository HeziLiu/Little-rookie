# 编写shell脚本，输出9×9表
## 要求
> 1.	在一页纸上打印代码及运行结果
> 2.  白底黑字
> 3.  乘法部分，必须使用函数完成（传递两个参数）
> 4. 第五行不输出
> 5. 输出表格的格式漂亮的话有+1

## 源代码
```
calc(){
	echo "$1*$2=$[ $1 * $2]"
}
for ((i=1; i < 10; i++)) {
	for ((j=1; j < 10; j++)) {
		if ((i == 5)) {
			break;
		}
		echo -n  `calc $i $j`'\t'
	}
	echo -n '\n'
}
```
